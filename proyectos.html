Aplicaciones Web

Parchapp


Tecnologías:


Parchapp es una aplicación que permite a las personas dar a conocer sus sitios favoritos para divertirse y asì dar a conocer esos sitios a otras personas que tal vez no sabìan de la existencia de estos.

Esta idea nació a partir de una observación que me hice acerca de las zonas donde vivimos, que a pesar de haber vivido toda nuestra vida en una ciudad, hay sitios que nos son desconocidos. y estoy seguro que a mas de uno nos pasa igual.

Bikeo


Tecnologías:


Bikeo está creado para poder acceder a una información más detallada de una bicicleta al momento de comprarla, ya que esta aplicación está diseñada para la venta exclusiva de bicicletas.

Bikeo al ser una plataforma que sólo se centra en bicicletas, permite a los vendedores y compradores tener un lugar más cómodo para hacer este tipo de transacciones.

Presupuesto


Tecnologías:


Esta aplicación fue hecha para que las personas pudieran observar de manera más gráfica sus ingresos y egresos, y así poder saber cómo está su economía.

Luego tuvo una modificación que permite ver la capacidad de endeudamiento de la persona. Esto fue realizado para que una persona distinta pudiera ingresar los datos de otra persona y así poder decidir acerca de realizar algún tramite con esta.

Demail (desarrollo)


Tecnologías:


Demail es una aplicacion de mensajería sencilla. Usa firebase como backend para facilitar los registros, las autenticaciones, y demás procesos del CRUD.
Está diseñada para servir como una aplicación de mensajería instantanea para una pequeña empresa que necesite una comunicación simple con sus empleados.
Actualmente se encuentra en desarrollo y está prevista para también funcionar como pwa para tener una mejor portabilidad y uso de esta.


Páginas Web

Ferretería


Esta página fue dearrollada con el fin de poder hacer visible una pequeña ferretería en internet.
La página consta de una sección principal en la que se trata sobre la ferretería y los servicios que ofrece, una seccion con información detallada de cada producto que tiene la ferretería, y, por último, una sección de contacto para que alguna persona que se vea interesada en algún producto o servicio se pueda poner en contacto con el establecimiento.

Sala de belleza


Esta página fue creada como material de estudio, simulando la necesidad de un establecimiento de tener su propia página web.
La página consta de secciones en la que se da una breve presentación sobre el sitio, otra donde se describe los procedimientos que tiene el establecimiento y finalmente una sección con las recomendaciones/opiniones de algunos clientes.
